# AutoForge Documentation

Welcome to the AutoForge documentation.

## Features

AutoForge is an autonomous coding agent system that:

- **Manages feature backlogs** using Convex for real-time state
- **Tracks project progress** with live dashboards
- **Schedules agent runs** for automated implementations
